# React Frontend Deployment

Most of the files are generated by react, we only need to modify [App.js](./src/App.js)

## Run exsiting Frontend

### Setup Server

```shell
npm install
```

### Run Server

```shell 
npm start
```

### Install new-package

```shell
npm install --save new-package
```
